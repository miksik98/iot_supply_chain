How to run

```
docker-compose up -d
npm install
nodejs src/application
```

Create table in database for producer to publish messageRootId product.
```
nodejs src/application
    - create producer
    - [company name]
```

Create product
```
nodejs src/application
    - create
```

Buy one product
```
nodejs src/application
    - buy one
```

Collect many products to create new product containing them as its compontents
```
nodejs src/application
    - buy many
```
